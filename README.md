# Todo List App

Простое приложение Todo-лист, созданное с использованием TypeScript и Vite.

## Возможности

- ✅ Добавление новых задач
- ✅ Отметка о выполнении (checkbox)
- ✅ Удаление задач
- ✅ Сохранение состояния (в памяти)
- ✅ Добавление по нажатию Enter

## Технологии

- TypeScript
- Vite
- HTML5
- CSS3

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/MaxuwkaPlus/НАЗВАНИЕ-РЕПОЗИТОРИЯ.git
```

2. Перейдите в папку проекта:
```bash
cd vite-todo-app
```

3. Установите зависимости:
```bash
npm install
```

4. Запустите проект в режиме разработки:
```bash
npm run dev
```

## Скрипты

npm run dev - запуск в режиме разработки
npm run build - сборка проекта
npm run preview - предпросмотр собранного проекта
